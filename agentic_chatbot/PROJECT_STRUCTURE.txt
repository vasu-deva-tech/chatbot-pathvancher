📋 WHAT REPLACED FROM N8N
────────────────────────────────────────────────────────────────────

N8N Component                    →  Agentic Equivalent
─────────────────────────────────────────────────────────────────
Webhook (entry point)            →  FastAPI /chat endpoint
Extract Session Info (JS code)   →  app/session.py
Lookup Session (Google Sheets)   →  SessionManager (Redis)
Create New Session (append)      →  session.create_session()
Extract Company Details (JS)     →  extract_company_details() tool
Buying Intent Detection (JS)     →  detect_buying_intent() tool
Get Knowledge Base (Supabase)    →  KnowledgeBase class
Generate Embedding (OpenAI)      →  EmbeddingService class
Smart Context Matching (complex) →  LangChain Agent
AI Assistant Response (OpenAI)   →  ChatOpenAI LLM
Format Response                  →  format_response() tool
Google Sheets Write              →  Optional (can re-add)

🏗️ ARCHITECTURE OVERVIEW
────────────────────────────────────────────────────────────────────

Request Flow:
────────────
1. User sends message via /chat endpoint
2. FastAPI validates request
3. SessionManager loads/creates session
4. EmbeddingService generates embedding
5. KnowledgeBase searches for matches
6. LangChain Agent decides:
   - Route to KB if high confidence
   - Route to LLM if low confidence
7. Agent executes tools as needed:
   - detect_buying_intent()
   - extract_company_details()
   - format_response()
8. Return JSON response
9. Update session history

Response Time Breakdown:
────────────────────────
- Embedding generation:    200-300ms
- Knowledge base search:   50-100ms
- Agent tools:             1-5ms
- LLM call:                100-200ms
- Total:                   200-500ms (vs 800-1200ms in n8n)
